# SuperbatGUI
Batch files and GUI interface for specific video archival purposes


Has dependencies outside .bat files and .au3 (needs to be compiled as .exe)
  - MediaConch (cli)
  - fsum (cli)
  - ffmpeg (cli)
  - ffprobe (cli)
  - ffplay (cli)
  - Graphics for GUI
  - QcTools
  - VirtualDub 2

Instructions on how to build will be posted at a later time
